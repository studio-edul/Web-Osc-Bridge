Web to TouchDesigner Bridge

TD PC의 로컬 IP와 포트 입력 (예: 192.168.0.100:9980)

처음 사용 시 (1회만):

  1. TD Web Server DAT → Port 9980, TLS 설정, Active On
  2. 모바일에서 https://[TD-IP]:9980 접속
  3. "연결이 비공개가 아닙니다" 경고 → 고급 → 계속
  4. 자동으로 이 앱으로 이동 + 주소 자동 입력됨

이후 접속: 바로 주소 입력 후 Connect
